# Pawtra Order Manager v1.0

Customer artwork review, revision and approval system for Pawtra.

## Included
- Customer order lookup by order number + email
- Artwork previews with signed URLs
- 3 free revisions
- Artwork approval
- Admin dashboard
- Shopify order webhook
- Supabase database and private storage integration
